0Claude API 設定

告知テキストの整形に Claude API を使います。キーはこのブラウザ（localStorage）にのみ保存され、送信先は api.anthropic.com のみです。※公開せず自分専用で使ってください。

1告知テキストを貼り付け → 自動抽出

店舗の公式LINE・X・Instagram などの告知文をそのまま貼り付けます。1つの投稿に複数トーナメントが含まれていてもOK。日付・大会名・バイイン・GTD等を自動抽出します。

2抽出結果を確認・編集 → 登録

内容を確認して必要なら修正し、「登録」で保存対象に追加します。抽出データは source: "semi" / verified: false で登録されます。
